My Apps and Projects

Hello, I'm a newbie developer. These are some applications and some projects that I've experimented on. I've mainly used vanilla JavaScript for these projects. Feel free to check them out.


Weather App

A beautiful app developed using vanilla JavaScript that gives you information about current weather of the location that you have entered.You can also set default location so that you dont have to enter it everytime you use the app.

Food Recipe Finder

A JavaScript app that searches the recipes of the meal according to your wish, displays ingredients required to make the meal and also save recipe as your favourite for future use.

Budget App

App to keep track of your daily incomes and expenditures and show detailed information on what item expenditure is more to make you self aware about your spending habits.

To-Do List App

As the name suggests, this app helps you to keep record of things that you have done or should be doing in a fancy manner. Data is stored in LocalStorage so no data is lost when you reload it.
